# Healthcare-AI-Driven
AuraHealth is an intelligent healthcare ecosystem simulation built with a "Patient-First" philosophy. Developed by me, it bridges the gap between reactive medical care and proactive clinical intelligence. The platform features role-based dashboards, a real-time facility locator, and an AI assistant designed to streamline the patient-doctor experience without the overhead of heavy frameworks.
# 🔬 The Vision
In modern healthcare, delays in communication can lead to critical outcomes. AuraHealth was developed to demonstrate how lightweight, high-performance web technologies can model complex medical workflows. As the Team Leader, I directed the development of AI Chatbot, Frontend, unified interface that supports patients, doctors, and administrators through a secure-simulated environment.

# 🛠 Technical Excellence
The project is a showcase of "Show, Don't Just Tell" engineering. Rather than relying on external libraries, we utilized Vanilla JavaScript to manage application state and localStorage for persistent user sessions. Key technical milestones include:

Geospatial Intelligence: Using Leaflet.js to map real-time healthcare facilities like SCB Medical College, providing users with instant directions.

Dynamic UI/UX: Implementing GSAP (GreenSock Animation Platform) to create fluid, high-stakes interfaces that remain responsive across all devices.

Intelligent Triage: A rule-based AI Assistant capable of emergency detection, automatically navigating users to urgent care facilities when critical keywords are detected.
